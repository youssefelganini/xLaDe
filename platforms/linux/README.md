## Evidence

Linux is the primary development platform for xLaDe.

Development, testing, debugging, documentation, and release preparation are routinely performed on Linux systems, meaning Linux compatibility is continuously verified throughout normal project development.

Unlike other platforms, which require dedicated validation records, Linux serves as the baseline environment for day-to-day development, testing, and release preparation.
