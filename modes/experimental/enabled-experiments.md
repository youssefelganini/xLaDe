# Enabled Experiments — Experimental Mode

The following experiments are enabled when xLaDe is used
in Experimental Mode:

- EXP-001: Enforced Proof Review
- EXP-002: Kernel Boundary Violation Detection
- EXP-003: Documentation Coverage Check

Experiments listed here may enforce policies, emit warnings,
or fail builds.

Stable and Onboarding modes MUST NOT enable these experiments.
